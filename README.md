Retrosmart Aurorae themes
=========================

A retrosmart-look collection of themes for KDE Aurorae.


Installation
------------

Install to `~/.local/share/aurorae/themes/` or to `/usr/share/aurorae/themes/`.

Create such directories if they don't exist.
